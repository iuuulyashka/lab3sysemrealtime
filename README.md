# Принцип работы:

Условие взятия монеты: каждый вор может взять монету только если у него ≤ монет чем у напарника

Чередование потоков: обеспечивается естественным образом через проверку баланса

Синхронизация: мьютекс защищает критическую секцию (проверка условий и изменение переменных)

# Для нечетных чисел:

Один вор получает на 1 монету больше

Разница всегда составляет ровно 1 монету

Примеры: 51/50, 4/3, 26/25



# Для четных чисел:

Монеты распределяются поровну

Разница составляет 0 монет

Примеры: 50/50, 3/3, 25/25
